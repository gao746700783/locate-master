# locate-master
  Android定位实现，包括原生定位、百度高德腾讯定位
## 实现详情
- 原生定位
    >LocationManager
- 百度定位
    >14年12月百度LBS开放平台发布定位SDK 5.0版本，首次开放全球定位能力
- 高德定位
    >14年8月开放全球定位LBS服务
- 腾讯定位
    >未成功
